# DESIGN DOCUMENT: Cyberpunk Tech Event Website

## 1. Design Philosophy

This platform should not feel like a traditional event website. It should feel like entering a **digital underground tech universe**.

The design direction is a hybrid of:

* Cyberpunk (Neon, dystopian, high-tech chaos)
* Controlled Neo-Brutalism (bold blocks, sharp edges, strong hierarchy)

The goal is to balance:

* **Visual aggression (cyberpunk)**
* **Structural clarity (brutalism)**

---

## 2. Core Visual Identity

### Color System

Primary Palette:

* Neon Yellow (#FCEE0A) → dominant surfaces (like reference)
* Deep Black (#0A0A0A) → base background
* Electric Cyan (#00F0FF) → highlights / hover
* Magenta (#FF007A) → accent interactions

Usage Strategy:

* Yellow = attention zones (hero, sections)
* Black = content grounding
* Cyan = interactive elements
* Magenta = rare highlights (don’t overuse)

---

### Typography

Primary Font Style:

* Condensed, bold, industrial
* Slight distortion / cyber feel

Examples:

* Headings: Wide, uppercase, aggressive spacing
* Body: Clean sans-serif for readability

Rules:

* Headings = ALL CAPS
* Tight line-height for impact
* Large font sizes for hierarchy

---

## 3. Layout System

### Structure Style:

* Grid-based but visually “broken”
* Sections feel like panels / terminals

### Key Characteristics:

* Hard edges (no soft rounded corners)
* Offset sections
* Layered overlays
* Asymmetry (intentional imbalance)

---

## 4. Hero Section (MOST IMPORTANT)

### Objective:

Make user feel:

> “This is not a normal event. This is something serious.”

### Design:

* Fullscreen animated background
* Cyberpunk characters / abstract tech visuals
* Overlay with noise / scanlines

### Content:

* Event Name (Huge)
* Tagline:

  > “ENTER THE SYSTEM”
* CTA Buttons:

  * Explore Events
  * Register Now

### Interaction:

* Slight parallax movement
* Hover glow effects
* Flicker animation (subtle)

---

## 5. Navigation

### Style:

* Minimal but sharp
* Transparent → becomes solid on scroll

### Items:

* Home
* Events
* Categories
* Register
* Admin (hidden)

### Behavior:

* Neon underline on hover
* Slide animation

---

## 6. Events Section (CORE FEATURE)

### Layout:

Grid of cards inspired by your references:

* Strong rectangular blocks
* Image + overlay text
* Category tag

### Categories:

* Robotics
* Computer Tech
* Puzzle / Logic
* School Level

---

### Event Card Design:

Each card includes:

* Title
* Category label (neon tag)
* Short description
* “View Details” CTA

### Interaction:

* Hover = glow + slight scale
* Border flicker animation

---

## 7. Category Sections

Each category should feel slightly different but consistent.

### Example:

#### Robotics:

* Cyan accents
* Mechanical textures

#### Computer:

* Grid + terminal vibe

#### Puzzle:

* Abstract patterns

#### School:

* Slightly softer but still cyberpunk

---

## 8. Event Detail Page

### Layout:

Left:

* Event image / poster

Right:

* Details:

  * Description
  * Rules
  * Time / Venue
  * Eligibility

Bottom:

* Register Button (BIG + glowing)

---

## 9. Registration UI

### Design Style:

* Terminal-like form

### Inputs:

* Neon borders
* Focus glow

### Feedback:

* Success = glitch animation + confirmation

---

## 10. Section Transitions

Inspired by your references:

* Hard cuts (not smooth fades)
* Diagonal separators
* Glitch transitions
* Noise overlays

---

## 11. Micro Interactions

* Button hover = neon glow + slight distortion
* Cards = lift + light flicker
* Cursor effects (optional)
* Loading = scanline animation

---

## 12. Motion Design

Keep it:

* Subtle but constant

Use:

* Parallax
* Hover animations
* Scroll-triggered reveals

Avoid:

* Over-animation (keep it sharp, not messy)

---

## 13. Brutalism Layer (Controlled)

Where to apply:

* Section dividers
* Cards
* Buttons

Characteristics:

* Thick borders
* Solid blocks
* Harsh spacing

BUT:

* Always overlay cyberpunk effects (glow, gradients)

---

## 14. Content Strategy

Tone:

* Bold
* Slightly aggressive
* Tech-heavy

Examples:

* “ENTER THE GRID”
* “CHOOSE YOUR DOMAIN”
* “REGISTER BEFORE SYSTEM LOCK”

---

## 15. Final UI Sections Flow

1. Hero (cyberpunk cinematic)
2. Category selector (bold panels)
3. Featured events
4. All events grid
5. CTA banner
6. Footer (dark, minimal, neon accents)

---

> “Make it feel like a product launch, not a college fest.”

Every section should feel:

* intentional
* high-budget
* immersive
